# Site Web de formation en anglais

E-learning web platform for English learning.

## Tech stack
- PHP
- JavaScript
- HTML/CSS
- MySQL

## Overview
This project is an e-learning web platform designed for English learning. It follows an MVC-style structure and includes course management, quizzes, videos, certificates and administration features.

## Features
- User management
- Course and content management
- Interactive quizzes
- Video-based learning content
- Certificate handling
- Administration features
